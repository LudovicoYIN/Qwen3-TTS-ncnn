# Samples

Small generated demo clips from the C++ dynamic frontend path.

Both files were generated by the `qwen3_tts` CLI with:

- C++ tokenizer and CustomVoice prompt construction
- dynamic talker prefill
- ncnn Vulkan runtime
- RTX 3090 on Windows
- `frames=25`, `sample_rate=24000`

| File | Text | Prompt length | Duration |
| --- | --- | ---: | ---: |
| `qwen3_tts_ncnn_short_en_dynamic.wav` | `Hello, welcome to Qwen text to speech.` | 21 | 2.0s |
| `qwen3_tts_ncnn_long_en_dynamic.wav` | `This is a longer demonstration sentence for the ncnn runtime...` | 55 | 2.0s |

These samples are intentionally short so the source repository stays small. The
runtime supports longer generation when the model package includes the dynamic
frontend assets described in `docs/MODEL_PACKAGE.md`.
